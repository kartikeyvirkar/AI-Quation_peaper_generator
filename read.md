# Packages to run the code

``` npm install express ```

```npm install dotenv ```

```npm install groq-sdk```

``` npm install ejs ```

## Optional

``` npm install --save-dev nodemon ```

#### Project Setup Example in ```package.json```

``` JSON
 {
  "scripts": {
    "start": "node index.js",
    "dev": "nodemon index.js"
  }
}

 ```

 ## TO RUN PROJECT

 ``` node server.js ```
 ###### OR
``` nodemon server.js ```
